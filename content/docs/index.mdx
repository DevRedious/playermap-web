Player Map builds a social gaming graph that gathers feedback and recommendation
data, giving studios powerful realtime analytics while empowering communities to
have a real impact on development.

You can install it **in less than 2 minutes**:

```bash
npm install player-map
```

In the current version, the **Boss Fighters** game is the entry point. If your
platform requires a different game as the entry point for the graph, follow the
**Configuration** section.

Feel free to contact us on [Discord](https://discord.com/invite/52mKT9g8NB) if
you need to. Your feedback and requirements will help us improve the product for
the gaming community.
