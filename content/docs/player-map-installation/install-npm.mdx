Install `player-map` in your project.

## Installation

### With pnpm

```bash
pnpm install player-map
```

### With npm

```bash
npm install player-map
```

### With yarn

```bash
yarn add player-map
```

## Installed version

The latest version of `player-map` will be installed. You can check the version
in your `package.json`:

```json
{
  "dependencies": {
    "player-map": "^1.0.18"
  }
}
```
